2. Criação de um Chatbot

Consiste na criação de um Chatbot(Similar ao GPT) que busca auxiliar empresas na utlização de banco de dados, por meio de uma LLM que se especializa em transformar linguagem de alto nível em requisições na linguagem de programação SQL, que por meio de linguagem Java será executada no banco de dados da empresa e devolverá, em linguagem de alto nível, a resposta para a requisição feita pelo usuário. Para realização do projeto utilizamos linguagem e interface Java com Java Swing, banco de dados MySQL SQLCoder, IA Ollama.
